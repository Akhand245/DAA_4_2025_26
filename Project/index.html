Algorithm Theory

Dijkstra's Algorithm

  • Greedy algorithm for weighted graphs
  • Finds shortest path from source to all vertices
  • Time Complexity: O(V²) or O((V+E)logV) with heap
  • Works only with non-negative weights

Bellman-Ford Algorithm

  • Dynamic programming approach
  • Handles negative weights (no negative cycles)
  • Time Complexity: O(VE)
  • Detects negative weight cycles

Algorithm Steps

1

Initialize

Set distance[source] = 0, others = ∞. Use priority queue for Dijkstra

2

Process Vertices

Extract minimum distance vertex, relax all adjacent edges

3

Relaxation

Update distances if shorter path found through current vertex

4

Complete

Repeat until all vertices processed or no updates
